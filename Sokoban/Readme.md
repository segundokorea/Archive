## [Sokoban (Ruby Quiz #5)](http://www.rubyquiz.com/quiz5.html)

Implementation of Sokoban by Hiroyuki Imabayashi. Uses a set of classic maps copyrighted by Thinking Rabbit. Simply run `soko` to start playing:

    $ ./soko
